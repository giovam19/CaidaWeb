# Caida
Venezuelan card game <br><br>

## Usage
To deploy it you will need to create a DB with the [squema.sql](https://github.com/giovam19/CaidaWeb/blob/main/db/squema.sql) file and fill the ```.env file``` with the access info for your DB.<br>
For example:<br><br>
```
DB_HOST="localhost"
DB_PORT="3306"
DB_USER="user"
DB_PASSWORD="*****"
DB_NAME="name"

```
### Execute
* For **Debug** can run the folowing command: ```npm run serve``` where you can make changes and see it refreshing **F5**<br>
* For **Start** can run the folowing command: ```npm run start```
